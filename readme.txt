Custom CSS for the WordCamp Europe 2016 theme.

== Description ==



CSS generated from this plugin needs to be copy and pasted into the `Appearance -> Edit CSS` screen.


[a plugin by Scott Evans](http://scott.ee/ "WordPress designer and developer")

== Installation ==

To install this plugin:

1. Upload the `wceu-2016-css` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!
